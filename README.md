# 📊 SQL para Análise de Dados

Este repositório contém exercícios e projetos práticos desenvolvidos durante meus estudos em SQL (MySQL), com foco em análise de dados e modelagem relacional.

🎯 Objetivo: consolidar conhecimentos em consultas, relacionamentos, agregações e manipulação de dados para atuação na área de Tecnologia e Dados.

---

## 🗂 Estrutura do Projeto

### 📁 database-structure
Scripts responsáveis pela criação das tabelas do banco fictício "banco", incluindo:

- clientes
- produtos
- categorias
- lojas
- locais
- pedidos

O modelo simula um cenário real de vendas e relacionamento entre entidades.

---

### 📁 queries

#### 1️⃣ Select, Order By e Where
Consultas básicas e filtros.

#### 2️⃣ Count, Group By e Having
Agregações e análise de dados.

#### 3️⃣ Join
Relacionamento entre múltiplas tabelas.

#### 4️⃣ Subquery
Consultas aninhadas para análises mais avançadas.

#### 5️⃣ Views
Criação de consultas reutilizáveis.

#### 6️⃣ Create, Insert, Update e Delete
Manipulação e controle de dados.
